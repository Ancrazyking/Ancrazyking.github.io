睡眠排序

给定一组元素集合,根据集合中的每一个元素,开启一个线程.线程睡眠的时间等于该元素的值.

  • Java 代码的实现(需要实现Runnable接口)
  • 睡眠排序
  1. 运行时间取决于最大元素的值

  2. 原理是构造n个线程,和元素中n个数一一对应.

    初始化后,线程们开始睡眠,睡眠完成后,则输出对应的元素.

  3. 当集合中的元素过多时,线程数量会很多,系统资源开销会很大.
